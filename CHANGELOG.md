# Open Reading 更新日志

本文件记录 Open Reading 各公开版本的主要变化。版本按发布时间倒序排列。

## v1.1.0 - 2026-07-17

Android 版本号：`1.1.0`，构建号：`10000`。

### 阅读与导航

- 新增书签功能，支持添加、删除、定位并跳转到书签位置。
- 重新设计阅读目录，加入目录与书签分页、章节搜索、当前章节定位和选中状态。
- 目录面板配色完整跟随当前阅读主题，包括背景、文字、按钮、选中项、拖动条和遮罩。
- 移除依赖 Material 字体的方框图标，目录标题、标签和操作改为更清晰的文字表达。
- 优化章节列表排版，取消重复的章节数量胶囊，使用更简洁的编号与当前章节提示。
- 上、下阅读边距可独立调整，并改善刘海、灵动岛和底部安全区域适配。

### 图书导入与存储

- 重构图书导入流程，支持多文件暂存、顺序导入、重复项提示、失败重试和结果汇总。
- Android 支持通过系统文件访问框架选择并持久授权书籍目录。
- iOS 增加本地 Files 与 iCloud Documents 的存储桥接和导入基础能力。
- 增加导入来源标识和数据库迁移，为目录扫描与云端文件物化提供稳定记录。

### 书源与搜索

- 拆分书源管理和书源搜索页面，改善启停、编辑、删除和搜索流程。
- 完善书源协议、示例和接口文档。

### 视觉与字体

- 更新 Open Reading 应用图标和阅读导航视觉。
- 扩充应用与阅读字体资源，并补充字体许可说明。

## v1.0.2

Android 正式发布构建号为 `6009`。

### 发现页

- 发现页专注书籍展示，移除页首介绍卡片和内嵌搜索面板。
- 推荐、分类和最新栏目分别缓存，切换栏目不再重复请求，并支持下拉刷新。
- 进入分类栏目后自动选中第一个分类并展示书籍。
- 书源增删或启停后自动刷新发现内容。

### 独立搜索页

- 搜索入口移动到发现页右上角，点击后进入全屏搜索并自动聚焦输入框。
- 支持聚合搜索全部已启用书源，或只搜索指定书源。
- 支持一键清除、滚动加载更多和失败重试。
- 书籍详情不再展示书源 ID、书籍 ID 等调试信息。

### 书架

- 书架搜索改为顶栏展开式搜索。
- “全部 / 在读 / 已读”筛选收纳到顶栏菜单。
- 修复玻璃效果下书架顶栏缺少毛玻璃质感的问题。

### 阅读设置

- 修复阅读设置面板无法通过顶部拖动区域下拉关闭的问题。
- 统一本地阅读器和在线书源阅读器的设置入口、翻页模式与字号范围。
- 字号、行距和边距在结束拖动后再应用排版，减少拖动卡顿。

## v1.0.1

### 发现与书源

- 新增发现体验，支持书源推荐、分类和最新内容浏览。
- 搜索结果支持自动加载下一页、手动加载更多和失败重试。
- 多书源分页独立维护状态，并在聚合结果时自动去重。

### 开源许可

- 从本版本起采用 GNU AGPL-3.0-only。
- `v1.0.0` 及更早版本已经获得的 MIT 权利继续有效。
- 新增授权边界、贡献许可和项目品牌说明。
- 更新用户协议与多语言许可文案。

## v1.0.0

- 跑通开放书源流程，支持将在线书籍加入书架、在线阅读和下载阅读。
- 新增多套阅读主题。
- 新增仿真翻页效果。
- 新增行间距调整。
- 优化阅读排版算法。

## v0.9.1

- 新增平板双页阅读布局与相关分页能力。
- 完善 Android、iOS、macOS、Windows 和 Linux 的跨平台发布准备。
- 增加 Android Release 自动签名和跨平台 GitHub Release 流程。
- 完善多语言发布内容、用户协议和项目网站信息。
- 改进原生 Flutter 阅读器的分页、排版、主题和安全保护。
