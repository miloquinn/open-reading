# iOS 26 App Icon Implementation Plan
> **For agentic workers:** REQUIRED SUB-SKILL: Execute these steps inline and verify each generated artifact before moving to the next step.

**Goal:** Turn the user-supplied Origo community icon into a correctly sized iOS 26 Icon Composer source and a legacy raster fallback.

**Architecture:** Remove only the connected light background from the supplied raster, isolate its two existing connected subjects into full-canvas PNG layers, and let Icon Composer provide appearance treatments. Preserve a flat resize of the original for Flutter UI and legacy iOS icon slots.

**Tech Stack:** Apple Icon Composer, ImageMagick, Xcode 27 asset compiler

## Global Constraints

- Do not redraw the supplied artwork.
- Preserve its composition, colors, and proportions.
- Do not modify Android icon sources or unrelated worktree changes.
