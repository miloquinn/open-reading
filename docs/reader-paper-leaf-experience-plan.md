# 纸页化阅读体验：设计与实施计划

## 目标

本轮改造只调整阅读页的呈现、抓图、手势物理和排版能力，不替换现有内容解析器、分页器或稳定阅读定位模型。

需要同时满足：

1. 正文、章节名、页码、时间和电量属于同一张 page leaf；横滑、无动画和仿真翻页复用同一 leaf。
2. 菜单与交互控制栏仍是 viewport HUD，不进入纸张抓图。
3. 仿真翻页保留现有圆柱卷页，并新增经典折页；两者共用手势判定、几何约束和弹簧状态机。
4. 快照使用语义缓存键，并以字节预算、按需预热和异步代际保护控制内存与首帧延迟。
5. 首行缩进和段落间距只作为显示排版投影；书签、进度和恢复位置继续使用原文 UTF-16 offset。
6. 经典折页采用 clean-room 几何与 shader 实现，只参考公开可观察行为和数学原理，不复制 LightInk 移植控制器或还原 shader。

## 代码检查结论

改造前存在三个耦合点：

- `ReaderChromeOverlay` 常驻绘制页码/进度，仿真翻页抓取的正文 `RepaintBoundary` 不包含它，因此纸张移动时状态文字固定在视口。
- 原翻页组件连续抓取 current/forward/backward 三张图，DPR 只设上限，没有语义缓存键、字节预算或异步截图失效保护。
- 原控制器把手势进度插值到目标点，圆柱 renderer 同时承担手势、抓图和动画状态，难以安全增加第二种纸张模型。

现有 `NativeTextPaginator` 的视觉行边界、fit 验证和正文 span 测量应保留。新增排版能力不能改用简化的高度二分分页器。

## 目标数据流

```text
canonical chapter text / styled blocks
              │
              ▼
ReaderTextLayout
  显示文字 + display boundary → source UTF-16 boundary
              │
              ▼
NativeTextPaginator
  仍负责视觉行边界与分页 fit
              │
              ▼
ReaderPaperPageLeaf
  正文 + 章节名 + 页码 + 时间/电量
              │
              ▼
ReaderPageSnapshot(page identity, layout fingerprint, theme, revision)
              │
              ▼
字节预算 LRU / 延迟预热 / generation 防竞态
              │
              ▼
共享手势几何 + SpringSimulation
        ┌─────┴─────┐
        ▼           ▼
   圆柱卷页      经典折页
```

## 分阶段实施

### 1. 完整纸页 leaf

- 新增统一的 `ReaderPaperPageLeaf`，正文和静态页脚在同一渲染子树内。
- 页脚包含本页自己的章节名、`pageNumber / pageCount`、分钟级时间和电量。
- 安全区计算继续为页脚保留高度，正文分页高度与最终绘制高度一致。
- 本地阅读器和在线书源的无动画、横滑、双页和仿真模式全部传入明确的页身份与本页元数据。
- 分页模式关闭 viewport 常驻状态；竖向滚动没有离散 page leaf，继续使用 viewport 状态。

验收：翻页中旧页脚随纸张离开，底层出现下一页自己的页脚；回弹时无钉屏文字或闪烁。

### 2. 快照身份与缓存

语义缓存键只由以下字段组成：

- 页面身份：书籍/书源、章节、页索引和稳定 source offset。
- 排版指纹：视口、字号、行高、边距、系统文字缩放、语言/方向、字体、首行缩进、段距和安全区签名。
- 主题：纸张与文字配色的主题 ID。

时间、电量等低频动态内容使用 `contentRevision` 使同一个语义 entry 更新，不把时间值写入 key，避免每分钟制造新 key。

缓存策略：

- 首帧永远先画 live leaf，截图在首帧后开始。
- 顺序为 current → forward → 延迟 backward；手势实际需要的方向可立即补抓。
- DPR 根据设备 DPR、2.5 上限和单张字节预算自适应。
- LRU 同时限制 entry 数量和估算 RGBA 字节数；内存压力时保留当前动画必需页并清理其余项。
- 每次页面集合、布局尺寸或内容 revision 改变都递增 generation；旧异步截图完成后直接释放，不能回填新页面。

验收：首帧无三页同步截图阻塞；改主题/字号后不会显示旧纹理；快速跨页时无异步串页；缓存预算可由单元测试计算验证。

### 3. 手势几何与真实弹簧

- 把左右翻页镜像到统一的“右边缘纸张”规范坐标系。
- 圆柱卷页仍按起手位置选择上角或下角；经典折页把起手高度映射为右侧连续边锚点，水平拖动形成近似竖直折线，明确从角落斜拉时才形成对角折页。
- 经典折线是固定纸角与触点的垂直平分线，与页面矩形求交得到折线端点。
- 中央区域先做横向意图判定，边缘区域可立即进入纸张拖拽；竖向或无可翻页方向的手势安全取消。
- 松手使用速度投影与当前进度共同决定完成或回弹。
- X/Y 坐标分别由 Flutter `SpringSimulation` 驱动；回弹与完成使用不同刚度/阻尼，但共用同一状态机。
- 点击、音量键和程序化翻页进入同一完成弹簧路径，回调每次只触发一次。
- 中央起手的位移门槛和横向意图比应足够小，使纸页在用户明确横拖后立即跟手；提交投影可略偏向快速甩动，避免短促手势先停顿再决定方向。
- 连续点击或音量键请求使用有界 FIFO 串行执行；上一页尚在收尾时不能静默丢弃下一次翻页，也不能因音量键自动连发形成无限积压。目标页快照在当前页之后立即预热，不额外等待一帧。
- 提交动画的视觉完成条件以 X 轴纸页完全离场为准；Y 轴只负责卷角轨迹，纸页已经离开屏幕后不再为了等待 Y 轴弹簧而保留旧页。页面切换回调返回后立即撤销仿真层，不再多等 `endOfFrame`。

验收：向前/向后几何镜像；折线端点始终落在页边；右向左水平拖动不被吸到右下角；斜向拖动连续跟随手指；慢拖可回弹，快速甩动可完成；程序化翻页与手势结果一致；快速连续触发不会丢页；新页出现时屏幕装订边不残留旧页阴影。

### 4. 双 renderer

- `ReaderPageMode.pageCurl` 仍代表“仿真翻页”，不扩充为新的分页模式。
- 新增独立 `ReaderPageTurnStyle.cylinder/classicFold`，切换样式不重新分页。
- 圆柱 renderer 继续使用现有双纹理圆柱 shader。
- 经典折页先绘制目标页，再对当前页纹理做平面反射采样，加入折痕明暗、纸背降对比和投影。
- 装订缝属于活动纸页的物理边界，只能随当前翻动纸页存在；提交完成并换成新页时，旧页装订缝必须与仿真层在同一帧撤销。
- 任一 shader 加载失败时应局部降级，不能让另一种样式失效；无 shader 时使用安全的平移 fallback。

验收：两种样式可在设置中切换并持久化；前后翻页方向正确；目标页、纸背和阴影层次无反向或残影。

### 5. 排版能力与稳定 offset

- `ReaderTextLayout` 把首行缩进规范为 0–4 个全角字宽，把段落间距规范为 0–2 个附加换行。
- 每个显示 UTF-16 boundary 映射回原文 boundary；生成的缩进/空行不会改变 canonical text。
- 在线书源分页把显示范围转换成原文 `startOffset/endOffset`。
- 本地富文本分页通过显示 run 重建原有 source span，保留标题、图片与正文样式。
- 分页测试不再用“显示页文字拼接等于原文”判断连续性；应检查首尾 offset、相邻 range 连续和恢复定位。

验收：缩进为 0、原文已有空格、空白段、CRLF、纯空白章节、图片页和跨章预览均不造成 offset 断裂。

### 6. 设置、平台状态与文档

- `ReaderSettingsStore` 持久化首行缩进、段落间距和仿真样式；非法旧值安全 clamp/fallback。
- 设置面板新增两个离散滑杆和仿真样式选择面板，本地/在线阅读器共用同一套键。
- Android/iOS 通过只读 method channel 提供电量百分比和充电状态；不支持的平台只显示时间。
- ARB 与生成的本地化代码同步更新。
- `structure.md` 记录模块边界，`Log.md` 记录验证结果和仍需真机调参的风险。

## 验证矩阵

自动化验证至少覆盖：

- geometry：上下角、前后镜像、边界交点、进度单调。
- leaf：章节名、页码、时间、电量均位于抓图边界内；snapshot key 的三个语义维度彼此独立。
- cache/controller：DPR/字节上限、前后程序化翻页、中央起手、完成与回弹回调次数。
- text mapping：已有缩进归一化、缩进为 0、段距生成字符、source range 连续。
- settings：默认值、非法值恢复、持久化、两个滑杆和两种仿真样式。
- reader integration：本地与在线阅读器的模式切换、跨章预览、图片/空白内容和稳定进度。
- 全项目 `flutter analyze`、全量 `flutter test` 和至少一个包含 runtime shader 编译的 debug 构建。

真机/模拟器观察重点：

- 慢速拖动时纸角、折线和手指是否保持可信关系。
- 深色/纯黑/羊皮纸主题下纸背与阴影是否过亮或脏污。
- 高 DPR 设备首次进入仿真模式时是否掉帧。
- 状态更新、旋转、改字号和快速连续翻页后是否出现旧页快照。

## 回滚边界

- 设置模型没有数据库迁移，只增加 SharedPreferences 键；旧版本会忽略它们。
- 删除经典折页 shader 注册并把样式固定为 `cylinder`，即可保留其余 leaf、缓存和排版改进。
- `ReaderTextLayout` 是 paginator 前的显示投影；可把缩进/段距默认改为 0，而不影响原文定位格式。
- page leaf 与 viewport HUD 分层明确，菜单逻辑不依赖翻页 renderer，可分别回滚。
